# v1.3.0

## 新功能

- **角色编号配置**：`config.yaml` 使用 `character: 0–4`（0 铁甲战士、1 静默猎手、2 故障机器人、3 亡灵契约师、4 储君）
- **一键安装向导**：`install.bat` / `scripts/sts2_setup_wizard.py` / `sts2 install-wizard`  
  选择宿主、Skill 目录、游戏路径、角色后自动完成 pip、模组、配置与 MCP

## 文档

- 统一中文角色译名（见 README 角色表）
- 简体中文 README 与集成说明
