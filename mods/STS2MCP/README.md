# STS2MCP 游戏模组（随仓库分发）

本目录为 [Gennadiyev/STS2MCP](https://github.com/Gennadiyev/STS2MCP) 发布的模组文件副本，便于离线安装。

| 文件 | 说明 |
|------|------|
| `STS2_MCP.dll` | 游戏模组主文件 |
| `STS2_MCP.json` | 模组清单 |
| `VERSION` | 对应上游 Release 标签 |

安装到游戏目录：

```text
<Slay the Spire 2>/mods/STS2_MCP.dll
<Slay the Spire 2>/mods/STS2_MCP.json
```

或使用 `python scripts/install_sts2_mcp_mod.py`、`sts2skill.exe` 自动复制。

上游更新时请同步替换本目录并更新 `VERSION`。
