# v1.0.6

## 版本策略

- 主版本固定 **1.0**，功能与修复以 **1.0.x** 补丁号递增（替代此前的 1.3.x / 插件 v2.x）。

## 变更

- 诊断：**`sts2 doctor`** 检查路径、STS2MCP 模组、API 端口、各宿主 MCP 配置；**`sts2 status`** 附带 doctor 摘要
- 文档：[docs/QUICKSTART.md](docs/QUICKSTART.md) 三分钟上手；README / README.en 链到 QUICKSTART
- Issue 模板：功能建议、宿主集成；`config.yml` 链到文档
- 安装：GUI **`sts2skill.exe`**（中英双语），内嵌 payload，选路径后解压到宿主目录、游戏 `mods/`、STS2_Skills 安装目录（与 v1.0.3 相同构建方式）

## 下载

| 文件 | 说明 |
|------|------|
| `STS2_Skills-v1.0.6.zip` | 源码（不含 exe） |
| `sts2skill.exe` | Windows GUI 安装程序（中英双语） |
